Limpador de Lista M3U

Instruções de uso:

  • Cole sua lista M3U no campo de texto OU faça upload do arquivo
  • No campo "Caracteres a remover", digite os textos que deseja eliminar (separados por vírgula)
  • Clique em "Processar Lista" para gerar a nova lista limpa
  • Use o botão "Copiar Lista" ou "Baixar Arquivo" para salvar o resultado

Exemplo: Para remover "BR: ", "BRA-" e "(M)(720)", digite: BR: , BRA-, (M)(720)
